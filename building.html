Requirements

ld2solr requires Java 7 or 8, and Apache Maven version 3 or above.

Build

Having procured a source tree from the ld2solr Github project, execute a normal Maven package build in the top directory: mvn package

You will then find an executable JAR in the target/ subdirectory.

Building the site

If you would like to have a local copy of this site, including JavaDocs, coverage reports, and other development paraphernalia, perform a build as described above, then perform a Maven site build: mvn site

You will find the generated documentation in target/site/.

Deploying the site

If you would like to have to publish the site to Github, perform a Maven post-site build: mvn post-site

You will have to have supplied appropriate credentials for UVaLib's Github account. See here for details.
